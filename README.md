# React App boilerplate
Sample biolerplate with react 16, webpack 4, babel, hmr, jest, eslint, prettier

### How to use
```
  git clone git@github.com:seballos/react-app-boilerplate.git
  cd react-app-boilerplate
  # run dev server
  yarn dev
  # build bundle
  yarn build
```
